# Vikash Goswami — Portfolio

Deploy to GitHub Pages:
1. Create a public repo on GitHub
2. Upload all files (index.html, style.css, script.js, README.md)
3. Settings → Pages → Branch: main → Save
4. Live at: https://your-username.github.io/repo-name/

File structure:
vikash-portfolio/
├── index.html
├── style.css
├── script.js
├── images/        ← drop your photo here as profile.jpg
└── README.md
